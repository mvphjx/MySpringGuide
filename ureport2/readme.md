# Getting Started

使用Springboot集成的ureport2，可打包为jar（独立应用程序部署）或war（部署到Tomcat）。
Tomcat支持8.0以上，7版本不支持，需要使用Spring+Spring mvc版本集成。


## URL 

####报表设计器URL
http://localhost:8080/ureport/designer
####预览报表URL
http://localhost:8080/ureport/preview?_u=报表名称
####导出pdf
http://localhost:8080/ureport/pdf/show?_u=报表名称
